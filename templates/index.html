How it works

The model uses a pretrained classifier (Resnet50) as a feature extractor. The extracted features are passed as input to a user-defined model, whose classifier is trained to detect 1 of 133 dog breeds. However, if it detects a human in the image it will try to classify the human's breed, otherwise, it returns an unknown category
